Blog Architecture

Although this blog has undergone serveral redesigns, this latest one is more minimalistic and inspired by the Google Developer Documentation Style Guide. The blog code itself is simple too, no frameworks or libraries - just good old fashioned vanilla JS. While Dev.to houses the articles, a custom Node script fetches all the articles, adds some formatting and meta tags and the generates HTML files to server here. The blog also uses the history push state method to create a psuedo-single page app to serve each article.
